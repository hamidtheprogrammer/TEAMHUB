# Mini Team & Project Management app

## Product Requirement.
TeamHub is a backend-only internal SaaS that centralizes team, project, and user management for an organization, providing secure access control, administrative oversight, and basic lifecycle communication via email

## Functional Requirements:
### Authentication
-	User registration, login, password reset.
### Authorization
-	User can access their team and project.
-	Admin can manage and assign users into teams.
-	Admin can create and delete projects.
-	Admin can assign a project to a team.
### User management
-	Admin can list, delete users.
### Team management
-	User has access to their team and projects
-	Admin can list, delete, and create teams.
-	Admin can assign users to teams and assign team leader.
### Project management
-	Admin can list, delete, and create projects.
-	Admin can assign a project to a team.
-	Admin can detach a team from a project.

## User stories
-	As a user, I want to be able to create and login to my account.
-	As a user, I want to be able to reset my password
-	As a user, I want to be able to see my teams.
-	As a user, I want to be able to see my team projects.
-	As an admin, I want to be able to login into my account.
-	As an admin, I want to be able to list and manage all users.
-	As an admin, I want to be able to list and manage all teams.
-	As an admin, I want to be able to assign users to a team.
-	As an admin, I want to be able to list and manage all projects.
-	As an admin, I want to be able to assign teams to projects.
